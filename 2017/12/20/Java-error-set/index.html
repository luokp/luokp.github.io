Java error-set《java compiler level does not match the version of the installed java project face》

  • 在不同eclipse环境中拷贝Java项目时，需要更改几个项目配置。

    1. 修改编译器版本设置，在Eclipse环境中，鼠标右键选择项目，点击Properties，选择Java Compiler，可以在窗口右边看到编译器版本，如图所示：

    2. 修改项目引入的系统jdk库中的jar包。

    3. web项目还要修改tomcat服务器版本设置。
  • 有时会出现错误“java compiler level does not match the version of the installed java project face”，解决办法：eclipse界面的工程中右击Properties-ProjectFacts-修改java版本可。
