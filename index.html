reverse pairs

一般规律

解决方案：基于BST、BIT、合并排序等，在这里，我想着重讨论这些解决方案的一般原则及其在一些类似问题中的可能应用。
核心思想：分解数组并解决子问题。定义：假设输入数组int nums[n],数组nums[i,j]表示从下标i到j（包括在内）的子数组，T[i,j]表示适用于这个子数组的同样问题。

有了上面的定义，我们直接给出初始的问题T[0,n-1],接下来的关键是如何从它的子数组来找出该问题的解决方案，本质上相当于构建T[i,j]的递推关系，如果能解决T[i,j]，就可以找出更大子数组直到解决初始问题T[0,n-1]。
引入两种常用地推关系：
1.顺序递推关系：T(i, j) = T(i, j - 1) + C,元素将顺序处理，C表示处理最后一个元素的子问题。
2.分区递推关系：T(i, j) = T(i, m) + T(m + 1, j) + C ，m = (i+j)/2，子数组nums[i,j]将被进一步划分为两部分，C表示结合两部分的子问题。

Java error-set《java compiler level does not match the version of the installed java project face》

  • 在不同eclipse环境中拷贝Java项目时，需要更改几个项目配置。

    1. 修改编译器版本设置，在Eclipse环境中，鼠标右键选择项目，点击Properties，选择Java Compiler，可以在窗口右边看到编译器版本，如图所示：

    2. 修改项目引入的系统jdk库中的jar包。

    3. web项目还要修改tomcat服务器版本设置。
  • 有时会出现错误“java compiler level does not match the version of the installed java project face”，解决办法：eclipse界面的工程中右击Properties-ProjectFacts-修改java版本可。
